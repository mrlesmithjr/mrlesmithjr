## Larry Smith Jr.

I work in IT and have been writing about infrastructure and automation since 2012. I started at Everything Should Be Virtual and now write at [Methodical Cloud](https://methodicalcloud.com). Over that time the work moved from doing the implementation to leading the teams and projects doing it, and the writing followed.

I move deliberately. I build a thing in a lab before I trust it anywhere that matters, I document what I learn, and I publish when I have earned the take. No hype, no schedule-driven content.

What I work in and write about: automation and configuration management with Ansible, virtualization and containers, and homelab infrastructure.
